# AKS template

This repository hosts an example AKS template consisting of:

* Azure Kubernetes Service
* Azure Container Registry
* Kubernetes / Docker / Docker Compose
* Terraform
* Node.js + Nest.js + TypeScript + Swagger + Prettier
